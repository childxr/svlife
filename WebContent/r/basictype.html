Basic Data Types

Overview

R有5种原子类型的object: character, numeric, integer, complex和logical。除了这5种类型以外，常用的对象是vector,vector里包含的元素全部都是同一类型的。空的vector可以用vector() command来建立。可以包含不同元素类型的数据类型是list。

Atrributes

R中的对象都会有以下几种属性

  • names, dimnames
  • dimensions
  • class
  • length
  • other user-defined attributes/metadata

如果想要获得对象的属性，可以使用attributes() command

  1.  x <- c(1,1,1,2,3)
  2. names(x) <- c("a","b","c","d","e")
  3. attributes(x)
  4. $names
  5. [1] "a" "b" "c" "d" "e"
