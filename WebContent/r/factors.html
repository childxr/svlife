Factors

Introduction

Factor是用来表示categorical数据的。它可以是有序或者无序的。在modelling的函数里，如lm()和glm()，factor会特别对待。用factor做标记会比用integer标记好，因为它更具有说明性。

Constructor

可以用factor() command来构造factor的向量。

单个参数
  • x <- factor(c("yes","yes","no","yes","no"))
  • x
  • [1] yes yes no  yes no 
  • Levels: no yes
两个参数（控制level输出的顺序）
  • x <- factor(c("yes","yes","no","yes","no"), levels=c("yes","no"))
  • x
  • [1] yes yes no  yes no 
  • Levels: yes no
