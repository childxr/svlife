Coersion

在R里，也有类型的自动转换，如可以将logical型的对象和integer型的对象连接在一起，结果生产的vector是integer型的，这种自动转换是implicit的。用不太准确的说法，如果将低级数据类型与高级数据类型相结合，那么结果低级数据类型将自动转换成高级类型；反之，则需要靠显示调用as.* command来完成转换。

logical < integer < numeric < character

factor < integer < numeric < character

logical 和 factor连接，将转换为integer

强制类型转换，调用as.numeric(), as.logical(), as.character(), as.factor(), as.complex()来转换。如果无法转换，如将"hello"转换为integer, 则会出现warning, 并用NA替代。
